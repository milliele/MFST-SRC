Action: RML/Backend/Scripts

RML/Backend/Scripts

do_action ('RML/Backend/Scripts')

This action is fired when all scripts and styles of Real Media Library are enqueued to WordPress. You do not need to worry about front-end functionlity. You also need not to worry about permissions, the action is only fired when the user is allowed to use Real Media Library.
See:
  • RML/Backend/Scripts/MediaLibrary for media library only scripts
Example

Enqueue a script

add_action( 'RML/Backend/Scripts', function() {
 wp_enqueue_script('my-rml-script', 'script.js', array('rml-general', 'rml-main'));
} );
