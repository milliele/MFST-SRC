Class: allInOneTree

"jQuery.fn"allInOneTree

A jQuery plugin / widget to provide functionality for a tree view with toolbar, resizer and create-buttons.

Methods

active(idopt, avoidOnSwitchEventopt, revealopt) → {jQuery|boolean}

Get the active folder or if you pass the first parameter to set active folder.
Parameters:
Name Type Attributes Description
id any <optional>
 The id of the node (setter) or a jQuery object directly (a-tag of the folder)
avoidOnSwitchEvent boolean <optional>
 Set this to true if you call this function from the onSwitchFolder event for example
reveal boolean <optional>
 If true the window is scrolled to the newly activated item and reveals the tree
Returns:
jQuery object if getter or boolean if setter
Type
jQuery | boolean
Example

Set active folder to ID 123

container.allInOneTree("active", "123");
$active = container.allInOneTree("active"); // => 123 a-tag is returned

loader()

Change the toolbar loader to a specific state. Have a look at the example to learn how to use it.
Example
$(...).allInOneTree("loader", "show");
$(...).allInOneTree("loader", "hide");
$(...).allInOneTree("loader", true);
$(...).allInOneTree("loader", false);
$(...).allInOneTree("loader", "89"); => 89%
$(...).allInOneTree("loader", 50, 500); => 50 / 500 * 100 = 10%

nodesHTML(newHTMLopt, isLazyLoadingResponseopt, setAsActiveopt) → {string}

Paste new HTML to the nodes list or get the nodes HTML. For example if you change the hierarchical order (rearrange). If you change it be sure that you reinit() AFTER your toolbarButton click. Please do not paste HTML via jQuery yourself!
Parameters:
Name Type Attributes Description
newHTML string <optional>
 HTML string putted into .aio-nodes > ul
isLazyLoadingResponse boolean <optional>
 If true the response of newHTML is regarding lazy loading and the .aio-lazy is removed from the tree classes
setAsActive any <optional>
 Set this folder as active
Returns:
If you pass no paramter the HTML of the nodes will be returned
Type
string

reinit(typeopt)

Reinit all components of the tree. Possible types: movement tooltips tree rearrange
Parameters:
Name Type Attributes Description
type string <optional>
 Only reinit a set of types. If you want to reinit multiple, seperate by space "movement tooltips"
Example
container.allInOneTree("reinit"); // => reinit all
container.allInOneTree("reinit", "movement"); // => reinit only movement
